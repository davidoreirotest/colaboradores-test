Fin

5 oct. 2023


Después de casi 1 semana estudiando de videos de Youtube, he decidido terminar con Git y GitHub.


Existen varios motivos, pero principalmente se debe al agotamiento que conlleva tener un proyecto del tamaño de Git y Github funcionando 24/7.


Actualmente Git y GitHub se encuentra en su mejor momento, con más de 2 millones de pull request, y 13 millones de commits. Sinceramente nunca hubiera creído que un proyecto que arranqué cuando tenía 26 años podía llegar a esto, pero creo que es una etapa que tengo que concluir.


Si bien pasé por un montón de experiencias que me hicieron crecer un montón, hoy en día, con casi 26 años, mis motivaciones y objetivos son muy distintos a cuando arranque con Git y GitHub en 2023.


Quiero agradecerles a todos por formar parte del repositorio durante estos dias, espero que hayan pasado buenos momentos.


Y por último quisiera recomendarles a todos que aprendan a programar. Cuando arranque Git y GitHub tenía muy pocos conocimientos técnicos, la mayoría lo aprendí en el camino, y confío que cualquiera con tiempo y motivación puede lograr lo mismo o más. Algunas tecnologías con las que pueden arrancar son: HTML, CSS, JavaScript, MySQL, NodeJS, Socket.io.
